# Setup WSL Ubuntu

## Ubuntu 26.04

```bash
wsl --install
```

```bash
wsl --list --online
```

```bash
wsl --install Ubuntu-26.04
```

```bash
wsl --unregister Ubuntu
```

```bash
wsl -d Ubuntu-26.04
```

```bash
nano ~/.bashrc
```

```bash
cd /home/sam
```

Press Ctrl + O then Enter to save, and Ctrl + X to exit
